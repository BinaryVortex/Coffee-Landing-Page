# Coffee Landing Page

A minimal, responsive Coffee Landing Page built using HTML and CSS.

![Landing Page Screenshot](./Screenshot%202024-07-03%20110806.png)

## About

This repository contains a simple, visually appealing landing page for a coffee brand. It demonstrates layout, typography, and basic responsive techniques using plain HTML and CSS — no frameworks or JavaScript required.

## Features

- Clean, modern design focused on product presentation
- Responsive layout that adapts to desktop and mobile
- Simple, easy-to-read HTML and well-structured CSS
- Ready to customize and use as a starting point for a small website

## Demo / Screenshot

The screenshot above shows the landing page as included in this repository. To view the live page, open `index.html` in your browser.

## Installation & Usage

1. Clone the repository:

   git clone https://github.com/BinaryVortex/Coffee-Landing-Page.git

2. Open `index.html` in your web browser:

   - Double-click `index.html`, or
   - Serve the folder with a simple HTTP server (optional):
     - Python 3: `python -m http.server 8000`
     - Then visit `http://localhost:8000`

## File Structure

- index.html — Main HTML file for the landing page
- styles.css — Main stylesheet (if present)
- Screenshot 2024-07-03 110806.png — Project screenshot used in this README

> Note: filenames with spaces are included exactly as they appear in the repository.

## Customize

- Replace text, images, and colors in `index.html` and `styles.css` to match your brand.
- Add more sections (menu, contact form, newsletter signup) as needed.
